  ```

  ```

- Java内存模型：

  ```java
  *栈:程序执行时所调用的方法（先进后出[出去的时候叫做弹栈]）
  *堆:程序执行时所创建的对象（new出来的东西都会进入堆内存，开辟出空间，并产生一份地址值）
  *方法区:方法没有被调用的时候，所在的内存（方法没有被调用的时候，就会放在方法区，等待被调用）
  -----------------------------------------------
  本地方法栈:调用操作系统底层相关的资源
  寄存器:交给CPU去使用的
  
  ```

- 方法后面.var   直接生成接收调用方法的变量和变量类型

  ```java
  p.getMax(12.3,11.1).var  
  //	生成：
  double max = p.getMax(12.3,11.1);
  ```
